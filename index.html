Pokemon Go WebApi

Retrieve and show data from Pokemon Go servers.

What is it ?

The target of this project is to provide a web interface to the Tejado's project, allowing to retrieve data from the Pokemon Go servers.

What can I do with it ?

This project allows simple users to generate "cards" exposing their progression ingame.

Moreover, we propose to developers a JSON endpoint to get data from the game.
