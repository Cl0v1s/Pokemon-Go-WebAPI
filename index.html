How can I use it ?

For Trainers

To get your trainer card, enter your login informations in the fields below. We will compute and encrypte them to produce some web code (HTML). Copy and paste this code where you want, it will show your trainer card.

*Please remember that your credentials are not saved, or used. Moreover they are encrypted before leaving your computer, we will never know them.

For developers

We also provide a JSON API endpoint to retrieve data from pokemonGo servers.

More informations are availables in the Github repo.
